# Engineering AI Assistant

An AI-powered **Retrieval-Augmented Generation (RAG)** application that enables users to ask questions about engineering documents and receive context-aware answers generated by an LLM.

The application combines a **Svelte frontend**, **FastAPI backend**, **ChromaDB vector database**, **Sentence Transformers embeddings**, and **Claude** to deliver grounded responses based on uploaded documents.

---

## Features

- AI-powered question answering
- Retrieval-Augmented Generation (RAG)
- Document embedding with Sentence Transformers
- ChromaDB vector database
- FastAPI REST API
- Svelte + Vite frontend
- Docker & Docker Compose support
- Support for engineering documentation and knowledge bases
- Extensible to PDF, TXT, Markdown, and other document formats

## Installation

### Clone the repository

```bash
git clone <repository-url>
cd engineering-ai-assistant-rag-sample
```

### Backend

```bash
cd backend

python -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt
```

### Frontend

```bash
cd frontend

npm install
```

---

## Configure Environment

Create a `.env` file:

```env
ANTHROPIC_API_KEY=your_api_key
```

---

## Run Locally

### Backend

```bash
cd backend

python ingest.py

uvicorn main:app --reload
```

Backend:

```
http://localhost:8000
```

Swagger:

```
http://localhost:8000/docs
```

### Frontend

```bash
cd frontend

npm run dev
```

Frontend:

```
http://localhost:5173
```

---

## Docker

Build the containers:

```bash
docker compose build
```

Run the application:

```bash
docker compose up
```

Stop the application:

```bash
docker compose down
```

---

## Knowledge Base

Place your documents inside the `documents/` directory.

Example:

```text
documents/
├── architecture.txt
├── BKK_Resume@AdamSiThuThetNaing.pdf
├── requirements.pdf
└── project_specification.txt
```

Re-index the knowledge base after adding new documents:

```bash
docker compose exec backend python ingest.py
```
